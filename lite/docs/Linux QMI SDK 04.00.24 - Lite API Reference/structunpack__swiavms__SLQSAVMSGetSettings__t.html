Detailed Description

Structure that contains the session type response for AVMS Set Settings command

Parameters
OMADMEnabled- OMA DM Enabled.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 1
Parameters
AutoConnect- Auto Connect.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 6
Parameters
AutoReboot- Auto Reboot.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 21
Parameters
FwAutodownload- Firmware auto download.
  • 0 - Disabled.
  • 1 - Enabled.
    • Bit to check in ParamPresenceMask - 3
FwAutoUpdate- Firmware auto update.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 4
Parameters
FwAutoSDM- OMA Automatic UI Alert Response.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 5
Parameters
pPollingTimer- Polling timer to connect to AVMS server.
  • 0-525600 (min)
  • 0:disabled
  • Bit to check in ParamPresenceMask - 16
Parameters
pConnectionRetryTimers- Connection Retry timers.
  • See PackSwiAVMSSettingsConnectionRetryTimers for more information.
  • Bit to check in ParamPresenceMask - 17
pAPNInfo- APN Information.
  • See PackSwiAVMSSettingsAPNInfo for more information.
  • Bit to check in ParamPresenceMask - 18
pNotificationStore- Notification Storing When Disabled or Offline.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 19
Parameters
pPeroidsInfo- Min and Max Period of an Observation.
  • See PackSwiAVMSSettingsPeriodsInfo for more information.
  • Bit to check in ParamPresenceMask - 20
resultcode- Tlv Result Code.
