Detailed Description

This structure contains MNC PCS Digit Include Status.

Parameters
MCC
  • A 16-bit integer representation of MCC.
  • Range - 0 to 999.
MNC
  • A 16-bit integer representation of MNC.
  • Range - 0 to 999.
MNCIncPCSDigit
  • This field is used to interpret the length of the corresponding MNC reported in the TLVs with an MNC.
  • Values
    • TRUE - MNC is a three-digit value. e.g., a reported value of 90 corresponds to an MNC value of 090
    • FALSE - MNC is a two-digit value. e.g., a reported value of 90 corresponds to an MNC value of 90
TLVPresent
  • TLV Present.
