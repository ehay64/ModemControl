Detailed Description

This structure contains information about the UMTS Info.

Parameters
umtsUarfcn
  • UTRA absolute RF channel number.
umtsPsc
  • Primary scrambling code.
umtsRscp
  • Received signal code power
  • The received power on one code measured in dBm on the primary CPICH channel of the neighbor/monitored cell.
umtsEcio
  • ECIO
  • The received energy per chip divided by the power density in the band measured in dBm on the primary CPICH channel of the neighbor/monitored cell.
umtsSqual
  • Squal
  • cell selection quality value in dB.
umtsSrxLvl
  • Srxlev
  • cell selection Rx level value in dB.
umtsRank
  • Rank of the cell.
umtsSet
  • Set of the cell.
