Detailed Description

This structure contains pack parameters to Set Audio Profile.

Parameters
Profile
  • Audio Profile
    • 0 - Handset
    • 1 - Headset
    • 2 - Car Kit
    • 3 - Speaker phone
    • 4 - Auxiliary
    • 5 - TTY
    • 6 - Auxiliary external PCM
    • 7 - Primary external PCM
    • 8 - External slave PCM
    • 9 - I2S
EarMute
  • Ear Mute Setting
    • 0 - unmuted
    • 1 - muted
MicMute
  • MIC Mute Setting
    • 0 - unmuted
    • 1 - muted
Generator
  • Audio Generator
    • 0 - Voice
    • 1 - Key Beep
    • 2 - MIDI
Volume
  • Audio Volume Level
    • 0 to 7
