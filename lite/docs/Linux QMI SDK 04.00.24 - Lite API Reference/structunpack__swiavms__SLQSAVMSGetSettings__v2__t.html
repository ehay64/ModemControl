Detailed Description

Structure that contains the response for AVMS Get Settings command It maps with SLQSAVMSSetSettings_v2 (For AVC2 service)

Parameters
OMADMEnabled- OMA DM Enabled.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 1
Parameters
AutoConnect- Auto Connect.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 6
Parameters
FwPromptdownload- Firmware Prompt download.
  • 0 - Disabled.
  • 1 - Enabled.
    • Bit to check in ParamPresenceMask - 3
FwPromptUpdate- Firmware Prompt update.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 4
Parameters
FwAutoSDM- OMA Automatic UI Alert Response.
  • 0 - Disabled.
  • 1 - Enabled.
  • Bit to check in ParamPresenceMask - 5
Parameters
pPollingTimer[OPTIONAL]- Polling timer to connect to AVMS server.
  • 0-525600 (min)
  • 0:disabled
    • Bit to check in ParamPresenceMask - 16
pConnectionRetryTimers[OPTIONAL]- Connection Retry timers.
  • See PackSwiAVMSSettingsConnectionRetryTimers for more information.
  • Bit to check in ParamPresenceMask - 17
pAPNInfo[OPTIONAL]- APN Information.
  • See PackSwiAVMSSettingsAPNInfo for more information.
  • Bit to check in ParamPresenceMask - 18
pNotificationStore[OPTIONAL]- Notification Storing When Disabled or Offline.
  • 0 - Disabled.
  • 1 - Enabled.
    • Bit to check in ParamPresenceMask - 19
pPeroidsInfo[OPTIONAL]- Min and Max Period of an Observation.
  • See PackSwiAVMSSettingsPeriodsInfo for more information.
  • Bit to check in ParamPresenceMask - 20
pAutoReboot[OPTIONAL]- Auto Reboot.
  • 0 - Disabled.
  • 1 - Enabled.
    • Bit to check in ParamPresenceMask - 21
resultcode- Tlv Result Code.
