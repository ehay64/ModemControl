Detailed Description

This structure contains the pack parameters for SLQSSetAudioPathConfig .

Parameters
Profile[Mandatory]
  • Audio Profile
    • 0-9
pECMode[Optional]
  • AV_EC
    • 0 - Echo cancellation off
    • 1 - Handset echo mode
    • 2 - Headset mode
    • 3 - Car kit mode
    • 4 - Speaker Mode
pNSEnable[Optional]
  • Noise Suppression
    • 0 - Noise suppression off
    • 1 - Noise suppression on
pTXGain[Optional]
  • TX Voice volume
    • 0x0000 - 0xffff
pDTMFTXGain[Optional]

    • 0x0000 - 0xffff
