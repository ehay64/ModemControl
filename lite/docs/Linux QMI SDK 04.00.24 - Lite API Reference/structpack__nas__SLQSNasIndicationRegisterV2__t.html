Detailed Description

This structure contains the SLQSNasIndicationRegisterExt request parameters.

Parameters
pSystemSelectionInd[Optional]
  • System Selection Preference indication registration. The following callbacks would not be invoked if the indication is disabled.
    tFNRoamingIndicator tFNDataCapabilities and tFNServingSystem
    • 0x00 - Disable
    • 0x01 - Enable
pDDTMInd[Optional]
  • DDTM (Data Dedicated Transmission Mode) indication registration. The following callbacks would not be invoked if the indication is disabled.
    tFNDDTM
    • 0x00 - Disable
    • 0x01 - Enable

    • 0x00 - Disable
    • 0x01 - Enable
