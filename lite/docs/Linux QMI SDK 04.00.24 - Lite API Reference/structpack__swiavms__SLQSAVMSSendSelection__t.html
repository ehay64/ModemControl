Detailed Description

Structure containing the AVMS selection

Parameters
selection[IN]
  • User Selection
    • 0x01 - Accept
    • 0x02 - Reject
    • 0x03 - Defer
pDeferTime[IN/OPTIONAL]
  • Defer time in minutes. A value of 0 will cause the prompt to be resent immediately.
pClientPerformOperationFlag[IN/OPTIONAL]
  • Client operation flag after accept.
    • 0: if modem performs the operation (download or update)
    • 1: if client performs the operation (download or update)
pPackageID[IN/OPTIONAL]
  • Package ID.
pRejectReason[IN/OPTIONAL]
  • Reject Reason.
