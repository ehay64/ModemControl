Detailed Description

This structure contains the unpack_dms_GetDeviceCapabilitiesV2 response parameters.

Parameters
DevCaps
  • See dms_devCaps for more information
  • Bit to check in ParamPresenceMask - 1
pDevSrvCaps
  • Device service capabilities
    • DMS_DEVICE_CAP_DATA_ONLY -0x01
    • DMS_DEVICE_CAP_VOICE_ONLY -0x02
    • DMS_DEVICE_CAP_SIMUL_VOICE_AND_DATA -0x03
    • DMS_DEVICE_CAP_NONSIMUL_VOICE_AND_DATA -0x04
  • Bit to check in ParamPresenceMask - 16
pDevVoiceCaps
  • Device voice capabilities
    • Bit 0 – GW CSFB
      • 0 – Not capable
      • 1 – Capable
    • Bit 1 – 1x CSFB
      • 0 – Not capable
      • 1 – Capable
    • Bit 2 – VoLTE
      • 0 – Not capable
      • 1 – Capable
  • Bit to check in ParamPresenceMask - 17
Parameters
pDevVoiceDataCaps
  • DevVoiceDataCaps
    • Bit 0 – SVLTE capability
    • Bit 1 – SVDO capability
    • Bit 2 – SGLTE capability
  • Bit to check in ParamPresenceMask - 18
pDevMultiSimCaps
  • See dms_devMultiSimCaps for more information
  • Bit to check in ParamPresenceMask - 19
pDevMultiSimVoiceDataCaps
  • See dms_devMultiSimVoiceDataCaps for more information
  • Bit to check in ParamPresenceMask - 20
pDevCurSubsCaps
  • See dms_devCurSubsCaps for more information
  • Bit to check in ParamPresenceMask - 21
pDevSubsVoiceDataCaps
  • See dms_devSubsVoiceDataCaps for more information
  • Bit to check in ParamPresenceMask - 22
pDevSubsFeatureModeCaps
  • See dms_devSubsFeatureModeCaps for more information
  • Bit to check in ParamPresenceMask - 23
pDevMaxActDataSubsCaps
  • Max number of subscriptions for data activity
  • Bit to check in ParamPresenceMask - 24
pDevMaxSubsCaps
  • See dms_devMaxSubsCaps for more information
  • Bit to check in ParamPresenceMask - 25
pDevMaxActDataSubsCaps
  • See dms_devMaxCfgListCaps for more information
  • Bit to check in ParamPresenceMask - 26
pDevExplicitCfgIndex
  • Explicit Cfg Index
    • (-1) - Modem controlled cfg
    • any other valid value
  • Bit to check in ParamPresenceMask - 27
Tlvresult
  • unpack Tlv Result
ParamPresenceMask
  • bitmask representation to indicate valid parameters.
